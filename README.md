# Blinkit-Sales-Analysis-Dashboard
This project provides a comprehensive analysis of Blinkit’s sales performance, customer satisfaction, and inventory distribution. The goal was to identify key insights and optimization opportunities by leveraging various KPIs and data visualizations in Power BI.

**Data Processing (Excel)**

Data Cleaning: Handled missing values and inconsistent naming conventions in the "Item Type" and "Fat Content" columns.
Data Transformation: Organized data into a structured format suitable for Power BI consumption.

**Dashboard Features & KPI's**
The dashboard tracks the following primary metrics directly from the source data:

Total Sales: ₹1.202M (Overall revenue generated).
Average Sales: ₹140.99 (Average revenue per sale).
No. of Items: 8,523 (Total count of different items sold).
Average Rating: 3.92 (Average customer rating).

**Key Insights**

Top Categories: Fruits, Vegetables, and Snack Foods are the highest revenue generators, each contributing approximately ₹0.18M.
Fat Content Impact: "Low Fat" items significantly outperform "Regular" fat items in total sales (₹776.3K vs ₹425.4K).
Outlet Performance: Supermarket Type 1 is the most successful outlet category, contributing over ₹787K in sales.
Location Trends: Tier 3 locations lead in sales volume compared to Tier 1 and Tier 2.

**Tools Used**

Microsoft Excel: For data cleaning and initial exploration.
Power BI: For data modeling, DAX calculations, and interactive dashboarding.
